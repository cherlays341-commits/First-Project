# Coffee Payment System (C++)

A lightweight command-line application that simulates a terminal payment system for a coffee machine. This project demonstrates my core understanding of conditional logic, user input handling, and basic arithmetic operations in C++.

## 📋 Features & Logic
The system processes three real-world payment scenarios:
1. **Exact Payment:** Validates if the user inserted the exact amount.
2. **Overpayment:** Calculates and returns the correct change to the user.
3. **Underpayment:** Detects insufficient funds and calculates the exact remaining amount due.

## 🛠️ Concepts Demonstrated
- Data Types (`int` for item count, `double` for currency/financial values)
- Input/Output Streams (`std::cin`, `std::cout`)
- Conditional Branches (`if`, `else if` statements)
- Basic Financial Math
